# Amazon-Combined-Data-Dashboard
Amazon Sales Dashboard built in Power BI showing order counts, revenue, reviews, shipment analysis, and category-wise trends. Includes bar charts, donut charts, map view, treemap, and yearly product description trends. Provides clear insights into top categories, sales performance, and customer engagement.
## Project Objective
To analyze Amazon sales data using Power BI, providing insights into orders, revenue, reviews, shipment regions, and product trends to support data-driven decision-making.
##Questions KPI 

Which product categories generate the highest number of orders?

What is the total sales revenue across all categories?

How many customer reviews were received overall?

Which shipment regions contribute the most to total revenue?

Which categories contribute the highest share of sales value?

How does product activity trend across years and quarters?

Which categories show the lowest sales and need improvement?

What is the relationship between reviews and product demand?

##Process
Process

Data Collection
Collected Amazon product, sales, shipment, and review data from the provided dataset.

Data Cleaning
Used Power Query to remove duplicates, fix data types, handle missing values, and format date fields.

Data Transformation
Created calculated columns, standardized categories, and prepared fields for price, reviews, and shipment analysis.

Data Modeling
Organized tables, built relationships, and structured the model for efficient visualization.

Dashboard Design
Designed a dark-theme Power BI dashboard with bar charts, donut charts, gauges, maps, and treemaps.

KPI Development
Identified key metrics like total revenue, total reviews, order count, and category-wise contribution.

Insight Generation
Interpreted visual outputs to identify sales patterns, top categories, shipment performance, and yearly trends.

Final Deployment
Published and documented the dashboard for GitHub, making it easy to view and understand.
